Крутите вниз, чтобы узнать больше!

O нас.

Цель проекта: Во фреймворке Gstreamer есть шейдеры для работы с видео и звуком. Наша задача состоит в том, чтобы оптимизировать работу шейдеров которые хранятся в формате .orc, с помощью SDIM-функций.

Задачи проекта:

  • 1. Первичные коммуникации с заказчиком. Получение технического задания.
  • 2. Разработка плана реализации проекта.
  • 3. Изучение алгоритма работы программы.
  • 4. Работа над алгоритмом оптимизации конвертации.
  • 5. Создание первой демо-версии и тестирование.
  • 6. Демонстрация демоверсии продукта заказчику.
  • 7. Доработка и оптимизация предполагаемого функционала.
  • 8. Оформление технической документации.
  • 9. Разработка текста презентации проекта.
